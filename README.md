# Calidad del Sueño

## Objetivo:
Explorar cómo se relaciona la calidad del sueño con el estilo de vida que lleva una persona, considerando elementos como su ocupación,
género, edad y estado de salud general.


## Dataset:
https://www.kaggle.com/datasets/uom190346a/sleep-health-and-lifestyle-dataset

## Tecnologías empleadas:
* Jupyter Notebook
* Pandas
* Matplotlib
* Seaborn
* PowerBI
